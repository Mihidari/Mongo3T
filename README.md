# Mongo3T

***
## Pré-requis
+ Node.js 10+
+ NPM 6+
+ MongoDB Community Edition 4.2

***
## Installation
Cloner le dépôt git dans votre répertoire de travail :
```console
$ git clone https://github.com/lescadev/Mongo3T.git
```

Installer les dépendances avec NPM :
```console
$ cd Mongo3T
$ git checkout dev
$ npm install
```
